# Sistem Manajemen Jadwal Mata Kuliah Berbasis Python

## 📌 Tentang Project
Project ini merupakan tugas kelompok mata kuliah **Algoritma dan Pemrograman II**.
Aplikasi ini dibuat menggunakan bahasa **Python** untuk membantu mengelola jadwal mata kuliah secara sederhana melalui terminal/console.

Fitur utama:
- Menampilkan seluruh jadwal mata kuliah
- Mencari jadwal berdasarkan hari
- Menambahkan mata kuliah baru
- Mengurutkan jadwal berdasarkan berbagai kategori
